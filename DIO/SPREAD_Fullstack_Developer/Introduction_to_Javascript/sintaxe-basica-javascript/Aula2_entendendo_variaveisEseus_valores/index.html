Como Declarar

var; variável global ou local - valor inicial nulo
let; variavel local de bloco - valor inicial nulo
const; variável local de bloco - valor inicial obrigatório

Definições de escopo GLOBAL e LOCAL

Usando var global

var nome = 'werlon';

nome = 'Werlon';

Usando var local

function nomeDaFuncao() {
var sobrenome = 'Lima';
console.log(sobrenome);
}
console.log(nome);
nomeDaFuncao();

As propriedades de var se aplicam a let, mas por convenção usamos let no escopo local

Declaração de variáveis

Existem 3 modos de declarar as variáveis em JavaScript:

var – escopo global e local, pode ter seu valor alterado, se não tiver um valor inicial será tratada como null;
let – escopo local de bloco, pode ter seu valor alterado, se não tiver um valor inicial será tratada como null;
const – escopo local de bloco, somente leitura, o valor inicial é obrigatório e não pode ser alterado.

Diferenças entre atribuição, comparação e comparação idêntica:

Comparação

var comparacao = "0" == 0;
console.log(comparacao);

Comparação Idêntica

var comparacaoIdentica = "0" === 0;
console.log(comparacaoIdentica);

Operadores Relacionais

Maior que

console.log(maiorQue);

Menor que

Maior ou igual a

Operadores Lógicos

&& - Todos os valores devem ser true

|| - Somente um valor deve ser true

! - Inverte os valores

var nao = !true;
console.log(nao);
